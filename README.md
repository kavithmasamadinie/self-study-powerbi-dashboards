1. Customer Shopping Behavior Analysis
File: customer_behavior_dashboard.pbix

Data Source: customer_shopping_behavior.csv (Retail transaction data)

Description: A comprehensive analysis of consumer spending habits to identify high-value customers and seasonal trends.

Key Insights:

Demographics: Purchasing power analysis broken down by Age, Gender, and Location.

Sales Trends: Seasonal performance (Winter vs. Summer) and top-selling categories (Clothing, Footwear, Accessories).

Customer Loyalty: Analysis of subscription impact, review ratings, and repeat purchase frequency.

Operations: Preferred payment methods and shipping type analysis.

2. Uber / Ride Booking Analysis
File: Uber.pbix

Data Source: ncr_ride_bookings.csv (Ride-booking dataset)

Description: An operational dashboard analyzing ride-booking data to optimize logistics, understand demand patterns, and improve service reliability.

Key Insights:

Demand Hotspots: Identifying peak locations for pick-ups and drop-offs.

Time Analysis: Visualizing peak hours to understand when demand surges vs. when cancellations occur.

Ride Status: Breakdown of completed vs. cancelled rides to identify service gaps.

Revenue: Fare analysis across different booking types and vehicle categories.

3. HR Insights Dashboard

Project Type: Workforce Analytics 

Description: An interactive dashboard designed to monitor workforce health and support data-driven HR decision-making.

Key Insights:


Attrition Analysis: Identifying turnover rates and trends to improve retention strategies.


Demographics: Visualizing employee distribution by age, gender, and department.


Performance: Tracking key performance indicators (KPIs) and department-wise headcount.


Features: Dynamic filtering and drill-down capabilities for resource planning.
